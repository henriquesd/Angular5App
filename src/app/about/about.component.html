Comands to use in command line:


Install the Angular CLI: $ npm install @angular/cli -g

Create an Angular project (like this example, using Sass styling and routing): $ ng new ProjectName --style=scss --routing


Open the project: $ cd ProjectName

Execute the project: $ ng serve --open


Creating a new Component: $ ng generate component home

Creating a new Component (other option): $ ng g c home


Install the Animation Library: $ npm install @angular/animations@latest --save


Create the Service File: $ ng generate service data
